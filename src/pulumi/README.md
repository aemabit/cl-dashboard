### PULUMI 

- List Stack: `pulumi stack ls`
- Select Stack: `pulumi stack select <STACK>`
- Create Stack: `pulumi stack init <STACK>`
- Config Region Stack: `pulumi config set aws:region <REGION>`

### AWS
